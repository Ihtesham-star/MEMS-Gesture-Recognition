# Contributing

## Project Status

This repository contains the complete source code from a **finished Master's thesis project** (defended April 2025, graduated June 2025).

## Purpose

This code is shared for:
- **Educational purposes**: Learning about gesture recognition and MEMS control
- **Research reference**: Understanding methodologies used in precision control systems
- **Code examples**: Demonstrating integration of computer vision with hardware control
